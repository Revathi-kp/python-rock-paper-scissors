# Rock, Paper, Scissors Game in Python

A clean, beginner-friendly implementation of the classic Rock, Paper, Scissors game built using Python 3. It runs directly inside your computer terminal and features infinite gameplay loops and continuous score tracking.

## 🚀 Features
- **Persistent Scoreboard:** Tracks your Wins, Losses, and Ties across multiple game sessions without resetting your score.
- **Input Cleaning:** Uses built-in string functions so that typing mistakes or capitals won't break or crash the game loop.
- **Looping Gameplay:** Automatically handles round resets and exits cleanly back to the terminal prompt when you type `n`.

## 🛠️ How to Run
Make sure you have Python installed on your computer. Open your system terminal or VS Code terminal, navigate to this project folder, and run:

```bash
python3 game.py
